# ЛР19 — Остов минимального веса (Kruskal и Prim)

Задание: в произвольном взвешенном связном графе найти остов минимального веса.

Реализованы два алгоритма:
- `mst_kruskal.py` — алгоритм Краскала (используется Union-Find / DSU).
- `mst_prim.py` — алгоритм Прима (реализован O(n^2) вариант, удобен для матрицы смежности).

Формат входа: граф задан взвешенной матрицей смежности `matrix`:
- `matrix[u][v] > 0` — вес ребра (u,v)
- `matrix[u][v] == 0` — ребра нет

Файлы:
- `mst_kruskal.py` — функция `kruskal(matrix)` возвращает `(edges, total_weight)`.
- `mst_prim.py` — функция `prim(matrix, start=0)` возвращает `(edges, total_weight)`.
- `interactive_mst.py` — демонстрация работы обоих алгоритмов.

Пример запуска:

```sh
python3 ЛР19/interactive_mst.py
```

Примечания:
- Оба алгоритма ожидают связный граф; если граф несвязный — будет выброшено исключение.
- Для больших и разреженных графов предпочтительнее реализация Прима через кучу или Краскал с эффективной сортировкой рёбер.
