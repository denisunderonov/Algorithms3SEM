# ЛР19: Минимальное остовное дерево (MST) - Инструкция по использованию

## Что такое MST?

**Минимальное остовное дерево** — это подграф, который:
- Соединяет все вершины
- Не содержит циклов
- Имеет минимальную сумму весов рёбер

Применение: прокладка кабелей, дорог, трубопроводов с минимальными затратами.

## Два алгоритма

### 1. Алгоритм Краскала

```bash
cd ЛР19
python3 mst_kruskal.py
```

**Идея:** Берём самые дешёвые рёбра, не создающие циклы.

### 2. Алгоритм Прима

```bash
cd ЛР19
python3 mst_prim.py
```

**Идея:** Растим дерево из одной вершины, добавляя самое дешёвое ребро.

## Режимы работы (для обоих алгоритмов)

### 1. Использовать пример графа
- Выберите вариант `1`
- Программа загрузит готовый неориентированный взвешенный граф из 5 вершин

### 2. Ввести свой граф
- Выберите вариант `2`
- Введите количество вершин
- Введите **симметричную** матрицу весов (для неориентированного графа)

## Пример ввода

Для графа из 3 вершин:
```
    0 --4-- 1
    |       |
    2       3
    |       |
    2 --5-- (треугольник)

Матрица смежности (симметричная!):
  0 4 2
  4 0 3
  2 3 0
```

**Важно:** Матрица должна быть симметричной (matrix[i][j] = matrix[j][i]), так как граф неориентированный.

## Что вы получите

Программа выведет:
1. Список рёбер, входящих в минимальное остовное дерево
2. Общий вес MST

Пример вывода:
```
Минимальное остовное дерево (рёбра):
  Ребро (0, 2) с весом 2
  Ребро (1, 2) с весом 3

Общий вес MST: 5
```

## Разница между алгоритмами

- **Краскал:** Сортирует все рёбра, выбирает по порядку (от меньшего к большему)
- **Прим:** Растит дерево из стартовой вершины, каждый раз добавляя ближайшую вершину

Оба дают **одинаковый результат** (один и тот же вес MST), но порядок рёбер может отличаться.
