# ЛР16 — Хеш-таблица

Задание: реализовать хеш-таблицу произвольного размера с обработкой коллизий. Реализованы вставка, удаление и поиск по индексу и по значению.

Файлы:
- `hash_table.py` — реализация класса `HashTable` (цепочки / chaining).
- `interactive_hash.py` — демонстрационный скрипт, показывающий вставку, поиск и удаление.

Короткое описание API `HashTable`:

- `HashTable(size=16)` — создать таблицу размера `size`.
- `insert(value)` — вставить значение, вернуть `(bucket_index, position)`.
- `search_by_value(value)` — вернуть список `(bucket_index, position)` для всех вхождений.
- `search_by_index(index)` — вернуть список значений в бакете `index`.
- `delete_by_value(value)` — удалить первое вхождение, вернуть `(index, pos)` или `None`.
- `delete_by_index(index, pos=None)` — если `pos` == None — очистить бакет и вернуть удалённые элементы, иначе удалить элемент по позиции.
- `display()` — печать компактного представления таблицы.

Пример запуска демо:

```sh
python3 ЛР16/interactive_hash.py
```

Примечания по реализации:
- Коллизии: цепочки (каждый бакет — список Python).
- Хеш: используется `hash(value) % size`.
- Удаление по индексу поддерживает очистку всего бакета или удаление элемента по позиции.

Дальнейшие улучшения (опционально):
- динамическое изменение размера (rehash) при большой нагрузке;
- поддержка пар (key, value) вместо одиночных значений;
- альтернативная стратегия разрешения коллизий (открытая адресация).
