# ЛР18: Алгоритм Дейкстры - Инструкция по использованию

## Запуск программы

```bash
cd ЛР18
python3 dijkstra.py
```

## Что делает программа

Программа находит **кратчайшие пути** от выбранной стартовой вершины до всех остальных вершин во **взвешенном графе** (где у рёбер есть веса).

## Режимы работы

### 1. Использовать пример графа
- Выберите вариант `1`
- Программа загрузит готовый взвешенный граф из 6 вершин

### 2. Ввести свой граф
- Выберите вариант `2`
- Введите количество вершин
- Введите матрицу весов рёбер построчно

## Пример ввода взвешенного графа

Для графа из 3 вершин:
```
    0 --5--> 1
    |        |
    3        2
    ↓        ↓
    2 --4--> (конец)

Матрица смежности:
  0 5 0
  0 0 2
  0 4 0
```

Значения:
- `0` = нет ребра
- Положительное число = вес ребра

## Что вы получите

Программа выведет:
1. Кратчайшие расстояния от стартовой вершины до всех остальных
2. Конкретные пути (последовательность вершин) с указанием длины

Пример вывода:
```
Кратчайшие расстояния от вершины 0:
  До вершины 0: 0
  До вершины 1: 7
  До вершины 2: 9

Пути от вершины 0:
  До вершины 1: 0 -> 1 (длина 7)
  До вершины 2: 0 -> 2 (длина 9)
```

## Важно

- Алгоритм Дейкстры работает **только для графов без отрицательных весов**
- Если вершина недостижима, программа укажет "недостижима"
