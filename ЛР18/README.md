# ЛР18 — Алгоритм Дейкстры

Задание: в произвольном взвешенном ориентированном графе найти кратчайший путь с помощью алгоритма Дейкстры. Граф задан взвешенной матрицей смежности.

Файлы:
- `dijkstra.py` — реализация алгоритма Дейкстры и функция для восстановления пути `reconstruct_path`.
- `interactive_dijkstra.py` — демонстрационный скрипт с примером матрицы и выводом расстояний/путей.

Формат матрицы:
- `matrix[u][v] > 0` — вес ребра u->v.
- `matrix[u][v] == 0` — ребра нет (в этой учебной реализации нулевой вес ребра не используется).

Пример запуска:

```sh
python3 ЛР18/interactive_dijkstra.py
```

Возможные улучшения (опционально):
- поддержка 0-весов и/или специального маркера (None / math.inf) для отсутствия ребра;
- проверка на наличие отрицательных ребер (для них нужен Bellman-Ford);
- чтение матрицы из файла и вывод путей в удобном виде (например, в формате edges или adjacency list).
