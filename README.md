# Algorithms and Data Structures (3rd Semester)

Учебный репозиторий с лабораторными работами по курсу **«Алгоритмы и структуры данных»**.

Содержит реализации базовых структур данных и алгоритмов на **Python**: сортировки, стек и очередь, списки, деревья, хеш-таблицы, графы, алгоритм Дейкстры, MST и задачу коммивояжёра.
