# ЛР17 — Графы (BFS и DFS)

Задание: реализовать обходы графа в ширину (BFS) и в глубину (DFS). Граф задаётся матрицей смежности.

Файлы:
- `graph.py` — класс `Graph` с методами `bfs`, `dfs` (рекурсивный), `dfs_iterative`, `neighbors`, `display`.
- `interactive_graph.py` — демонстрация работы обходов на примере ориентированного и неориентированного графа.

Инструкция по запуску:

```sh
python3 ЛР17/interactive_graph.py
```

Кратко о реализации:
- В `Graph` принимается квадратная матрица смежности (список списков). Значение != 0 означает наличие ребра.
- `bfs(start)` возвращает список вершин в порядке обхода в ширину от `start`.
- `dfs(start)` — рекурсивный обход в глубину от `start`.
- `dfs_iterative(start)` — итеративная версия DFS с использованием стека.

Возможные улучшения (опционально):
- обход всех компонент связности (если граф несвязный);
- поиск кратчайших путей в невзвешенном графе (BFS тоже возвращает расстояния);
- представление графа списком смежности для больших и разреженных графов.
