# ЛР17: Графы и обходы - Инструкция по использованию

## Запуск программы

```bash
cd ЛР17
python3 graph.py
```

## Что делает программа

Программа позволяет создать граф и выполнить три вида обходов:
- **BFS** (обход в ширину) — идём слоями, используя очередь
- **DFS рекурсивный** — идём в глубину через рекурсию
- **DFS итеративный** — идём в глубину через стек

## Режимы работы

### 1. Использовать пример графа
- Выберите вариант `1`
- Программа загрузит готовый граф из 6 вершин
- Вам останется только выбрать стартовую вершину для обходов

### 2. Ввести свой граф
- Выберите вариант `2`
- Введите количество вершин
- Укажите, ориентированный ли граф (да/нет)
- Введите матрицу смежности построчно

## Пример ввода матрицы

Для графа из 3 вершин:
```
Вершины: 0 -- 1 -- 2

Матрица смежности:
  0 1 0
  1 0 1
  0 1 0
```

Это означает:
- Из вершины 0 есть ребро в вершину 1
- Из вершины 1 есть рёбра в вершины 0 и 2
- Из вершины 2 есть ребро в вершину 1

## Что вы получите

После ввода программа выведет:
1. Матрицу смежности графа
2. Результат BFS обхода
3. Результат DFS (рекурсивного) обхода
4. Результат DFS (итеративного) обхода

Все результаты показывают порядок посещения вершин.
