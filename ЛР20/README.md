# ЛР20 — Задача коммивояжёра (TSP) — Held‑Karp

Задача: найти гамильтонов цикл минимального веса в взвешенном графе.

Реализация:
- `tsp_held_karp.py` — точный алгоритм Held‑Karp (динамическое программирование по битмаске).
  - Сложность: O(n^2 * 2^n) по времени, O(n * 2^n) по памяти. Подходит для n ≲ 20 (на практике n ≤ 12–16 комфортно).
  - Функция `held_karp(matrix, start=0)` возвращает `(cost, path)`, где path — цикл от start к start.
- `interactive_tsp.py` — демонстрация на маленькой симметричной матрице расстояний.

Формат матрицы:
- `matrix[i][j]` — вес (расстояние) от i до j. Для отсутствия ребра используйте `math.inf` или положительное большое число.
- В учебном варианте 0 (вне диагонали) трактуется как отсутствие ребра и заменяется на `inf` при запуске.

Компромиссы и улучшения:
- Для больших n используйте приближённые алгоритмы (nearest neighbor, 2‑opt) или ветвление‑и‑границу.
- Held‑Karp даёт точное решение, но экспоненциальен по сложности.

Пример запуска:

```sh
python3 ЛР20/interactive_tsp.py
```
